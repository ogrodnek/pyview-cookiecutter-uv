# Pyview Cookiecutter Template (uv)

This is a [cookiecutter](https://github.com/cookiecutter/cookiecutter) template for creating a new [pyview](https://github.com/ogrodnek/pyview) project using [uv](https://docs.astral.sh/uv/) for dependency management and including [Web Awesome](https://webawesome.com/) component library and design system.

## Features

- **uv**: Fast, modern Python package management
- **Web Awesome**: modern web component library and design system

## Prerequisites

- [cookiecutter](https://github.com/cookiecutter/cookiecutter)
- [uv](https://docs.astral.sh/uv/) (will be used in generated project)

## Usage

```sh
cookiecutter gh:ogrodnek/pyview-cookiecutter-uv
```

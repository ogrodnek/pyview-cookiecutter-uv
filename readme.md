# pyview-cookiecutter-uv

A [cookiecutter](https://github.com/cookiecutter/cookiecutter) template for [pyview](https://pyview.rocks) â€” build real-time web apps in Python with server-rendered HTML and WebSocket updates.

This template uses [uv](https://docs.astral.sh/uv/) for fast dependency management and includes [Web Awesome](https://webawesome.com/) for ready-to-use components.

## What You Get

- A working counter example to start from
- [uv](https://docs.astral.sh/uv/) for fast dependency management
- [Web Awesome](https://webawesome.com/) component library
- [just](https://github.com/casey/just) commands for common tasks
- Dockerfile (for deployment)
- Hot reloading for development

## Usage

```sh
cookiecutter gh:ogrodnek/pyview-cookiecutter-uv
```
